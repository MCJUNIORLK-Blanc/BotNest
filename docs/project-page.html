🎮 BotCommander

The ultimate Discord bot management panel with gaming-themed interface, real-time monitoring, and one-command installation.

🚀 One-Command Installation

Install BotCommander on your Linux server in seconds:

curl -fsSL https://raw.githubusercontent.com/MCJUNIORLK-Blanc/TestDCBotPanel/main/install.sh | sudo bash

With custom domain and SSL:

curl -fsSL https://raw.githubusercontent.com/MCJUNIORLK-Blanc/TestDCBotPanel/main/install.sh | sudo bash -s -- \
--domain your-domain.com \
--email your-email@domain.com \
--ssl

✨ Features

🎮
Gaming-Themed Interface
Dark gaming aesthetic with purple/blue accents, intuitive navigation, and responsive design for desktop and mobile.
🤖
Multi-Language Bot Support
Support for Node.js and Python Discord bots with multiple templates: Basic, Music, Moderation, and Economy bots.
📊
Real-Time Monitoring
Live bot status monitoring, system resource tracking (CPU, Memory, Disk), and WebSocket-powered real-time updates.
📁
Built-in File Manager
Code editor with syntax highlighting, file upload/download capabilities, and complete directory structure management.
🔧
Process Management
PM2 integration, automatic bot restarts, one-click start/stop controls, and comprehensive logging system.
🌐
Apache Integration
Reverse proxy configuration, SSL/TLS support with Let's Encrypt, static file serving, and WebSocket proxying.
